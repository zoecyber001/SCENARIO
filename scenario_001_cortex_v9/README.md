## üöÄ Quick Start

### Prerequisites
- Python 3.x
- `textual` - TUI framework
- `edge-tts` - Text-to-speech
- `mpg123` - Audio playback

### Installation

```bash
# Install Python dependencies
pip install textual edge-tts

# Install audio playback
sudo apt install mpg123  # Linux
brew install mpg123      # macOS
```

### Run

```bash
./run.sh
```

Or directly:
```bash
python3 cortex_v9.py
```
